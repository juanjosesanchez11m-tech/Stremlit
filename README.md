## 📝 Descripción del Proyecto
Esta aplicación es un tablero interactivo diseñado para el curso de **Talentotech**. Su objetivo es facilitar el análisis exploratorio de datos (EDA) sobre la pandemia en Perú durante agosto de 2020, integrando visualizaciones profesionales y documentación técnica en una sola interfaz.

## 📂 Estructura del Repositorio
* `app.py`: Código fuente de la aplicación Streamlit.
* `requirements.txt`: Lista de dependencias del proyecto.
* `README.md`: Guía de usuario y documentación técnica.
